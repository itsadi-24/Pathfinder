# Pathfinding Visualizer

A visually stunning pathfinding visualization app built with **React**, **Tailwind CSS**, and **Framer Motion**. It allows users to see how different pathfinding algorithms like DFS, BFS, and Dijkstra's Algorithm work in real time.

- **Depth-First Search (DFS)**: Explores as far as possible along each branch before backtracking.
